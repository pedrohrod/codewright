# Performance-test interpretation

Last verified: 2026-07-22.

- Derive pass/fail thresholds from service objectives when available.
- Report percentile latency, throughput, and error rate together.
- Tag scenarios and endpoints so aggregate results remain diagnosable.
- Treat a threshold failure as a failed test, not a warning.

Primary source: https://grafana.com/docs/k6/latest/using-k6/thresholds/
